# GameSpy 3D Master Servers List
This is a working list of game master servers that are compatible with GameSpy 3D. While many games that relied on GameSpy have had their own aftermarket master servers through updates or fan patches and even work with GameSpy Arcade, not all of them are compatible with the original GameSpy 3D.

### Disclaimer
I do not own every game that is supported by Gamespy 3D, so this is not a comprehensive list. If you know of a GS3D-compatible server list that isn't included here, make an issue and submit the info there.

### Server List Types
**Master Server** - This is a regular master server that can be added for a game. Not all games can have master servers added and instead must rely on the other types of server lists.

**SmartSpy List** - For games that don't allow for adding regular master servers, it may be possible to add them as a SmartSpy master server. This allows for adding filters for what games are shown when fetching the list from the server. Some SmartSpy lists may need the **LANMaster Server** option to be checked in order for GameSpy 3D to read them properly.

**Server List** - This type allows a list of servers to be pulled from a web URL (HTTP only, no HTTPS) or a file.

### Server IP Format Key

Server List Name - Address:Port (extra info)
